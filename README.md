# Week 9 Example 3 — Higher or Lower


## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

Click **Higher ▲** or **Lower ▼** to guess. Reach the target score shown in the HUD to advance to the next level. Equal cards count as wrong — you must commit to a direction.

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`

## Debug Controls

Press **D** to toggle the debug panel.

Available shortcuts:

- **1** = Jump to Level 1
- **2** = Jump to Level 2
- **3** = Jump to Level 3
- **S** = Go to Start Screen
- **W** = Go to Win Screen
- **O** = Go to Game Over Screen

The debug panel displays these shortcuts in game and can be shown or hidden at any time by pressing **D**.
